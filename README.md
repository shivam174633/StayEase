# StayEase – Property Rental Management System

StayEase is a full-stack property rental management platform that allows users to create, manage, and explore property listings with ease. It supports authentication, CRUD operations, interactive maps, and dynamic server-rendered pages for a seamless rental experience.

---

##  Live Demo

 https://stayease-ot88.onrender.com

---

## ✨ Features

* 🔐 User Authentication (Login / Register)
* 🏠 Create, Edit & Delete Property Listings
* 📋 View All Available Properties
* 🧾 Detailed Property Pages
* 🗺️ Interactive Maps using Leaflet.js
* 🔎 Location-based property visualization
* 🔒 Authorization (Only owners can edit/delete listings)
* ⚡ Server-side rendered dynamic pages
* 🗄️ MongoDB Atlas integration
* ⭐ Reviews & ratings system

# 🗺️ Leaflet Maps Feature

StayEase uses Leaflet.js to:

Display property locations on interactive maps
Show exact coordinates of listings
Help users visually explore properties

# 🔄 Workflow
User registers / logs in (Passport.js authentication)
User creates property listing with location data
Data stored in MongoDB Atlas
Listings rendered using EJS
Leaflet maps show property location
Authorized users can edit/delete listings
